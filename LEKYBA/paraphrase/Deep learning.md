Deep learning is a subset of Machine learning concept.
Deep learning use multilayered [[Neural Network]]s (called deep neural networks) that more closely simulate the complex decision-making power of human brain. Meanwhile it is simulating the human behavior in making decision.

Each has at least 3 or more hidden neural network layers. While classic machine learning just have 1 or 2.
