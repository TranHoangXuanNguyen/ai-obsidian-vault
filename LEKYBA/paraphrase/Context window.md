> [!NOTE] Context window
> The context window (or “context length”) of a [large language model (LLM)](https://www.ibm.com/think/topics/large-language-models) is the amount of text, in [tokens](https://www.ibm.com/docs/en/watsonx/saas?topic=solutions-tokens), that the model can consider or “remember” at any one time. A larger context window enables an [AI model](https://www.ibm.com/think/topics/ai-model) to process longer inputs and incorporate a greater amount of information into each output.
